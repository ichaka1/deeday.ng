# deeday.ng — Project Plan

**Type:** AI-powered Nigerian Event Management Platform  
**Domain:** deeday.ng  
**Status:** Planning Phase  
**Started:** 2026-04-08

---

## Vision

A two-sided event platform built specifically for Nigerians. Not generic watercolor templates copied from Zola or Knot. Real, culturally resonant designs — Aso-oke patterns, Ankara prints, rich colours, gold foil aesthetics — generated by AI and served through smart, functional invite websites.

Two sides:
- **The Day** (Wedding) — smart invitation websites with RSVP, gifting, and automated reminders
- **The Memory** (Burial) — AI-generated burial brochures, tributes, and memorial pages

---

## SIDE 1: Wedding (Priority Build)

### User Flow Overview

```
Couple creates event on deeday.ng
  → Gets a unique shareable link: deeday.ng/invite/[slug]
  → Shares link via WhatsApp, SMS, email

Guest visits the link
  → Sees a beautiful, culturally Nigerian invite page
  → Two buttons: [Attending] [Not Attending]

--- NOT ATTENDING PATH ---
  → "We understand. Would you like to support the couple?"
  → Suggested amounts shown (e.g. ₦5,000 / ₦10,000 / ₦20,000 / Custom)
  → Payment options:
      - Pay via Paystack (card, bank transfer, USSD)
      - OR see couple's account number to transfer manually
  → Confirmation message shown

--- ATTENDING PATH ---
  → Short form:
      - Name / Nickname (what the couple calls them)
      - Email address
      - Phone number
      - Number of guests coming (optional)
      - Dietary needs / Special note (optional)
  → On submit:
      - Guest added to MailerLite list for this event
      - MailerLite automation triggers 3 reminder emails:
          1. One week before wedding
          2. Two days before
          3. Morning of the wedding
      - "Add to Calendar" shown: Google Calendar / Apple Calendar / Outlook
      - Confirmation page shown with event details
```

## Core Features Breakdown

### 1. Couple Dashboard
- Create event (wedding or burial)
- Upload photos (couple's photo, venue photos)
- Set wedding date, time, venue, dress code
- Set gifting options (Paystack key or account number)
- Choose invite design (AI-generated from Nigerian styles)
- View guest list in real time (RSVP tracker)
- See who paid and how much (gift tracker)
- Download guest list as CSV

### 2. AI Invite Design Generator
- Couple picks a style category:
  - Yoruba traditional (Aso-oke, royal tones)
  - Igbo traditional (George fabric, green/gold)
  - Hausa/Fulani traditional (embroidery, arches)
  - Modern Nigerian (Afrobeats-era, vibrant, editorial)
  - Christian church wedding
  - Muslim Nikah
- They enter names, date, venue, theme colour
- AI generates a full-page invite design
- Used as hero section of their invite website

### 3. RSVP Website (Per Event)
- Custom URL: `deeday.ng/[coupleslug]` or custom domain (future)
- Mobile-first (99% of Nigerian guests will visit on phone)
- Fast loading even on 3G
- WhatsApp-shareable preview (OG image generated per couple)

### 4. Payments (Gifting)
- Paystack integration (primary — Nigerian default)
- Manual bank transfer option (account name, number, bank)
- Gift tracker on dashboard shows who paid what
- Optional: anonymous gifting option

### 5. MailerLite Integration
- Each event creates a MailerLite group
- Attending guests auto-subscribed to that group
- Pre-built automation:
  - Email 1: 7 days before — "It's almost time! Here's everything you need to know"
  - Email 2: 2 days before — "See you in 2 days! Venue details + dress code reminder"
  - Email 3: Morning of — "Today's the day! [Couple names] are getting married today"
- Each email includes Add to Calendar button

### 6. Calendar Integration
- Google Calendar: deep link that pre-fills event
- Apple Calendar: .ics file download
- Outlook: .ics file works here too
- WhatsApp reminder option (future phase)

## Tech Stack (Recommended)

| Layer | Choice | Reason |
|---|---|---|
| Frontend | Next.js 14 (App Router) | SSR for invite pages, fast, SEO-able |
| Styling | Tailwind CSS | Fast to build, easy to theme per couple |
| Backend/API | Next.js API Routes or Node/Express | Keep it simple to start |
| Database | PostgreSQL (via Supabase) | Free tier, auth included, real-time |
| Auth | Supabase Auth | Email/Google login for couples |
| Payments | Paystack | Nigerian standard, easy API |
| Email | MailerLite API | Already used in other projects |
| AI Images | fal.ai or Together AI (FLUX) | Cheap, fast, customizable prompts |
| File Storage | Supabase Storage or Cloudflare R2 | For couple photos, generated images |
| Hosting | Vercel | Free tier, Next.js native |
| Domain | deeday.ng | .ng domain, register on Whogohost/Namecheap |

## Database Schema (Draft)

```sql
-- Couples / Event creators
users (id, email, name, phone, created_at)

-- Events
events (
  id, user_id, slug, type [wedding|burial],
  title, date, time, venue, dress_code,
  bride_name, groom_name, couple_photo_url,
  design_style, invite_image_url,
  paystack_key, bank_name, account_number, account_name,
  mailerlite_group_id,
  created_at
)

-- Guests
guests (
  id, event_id, name, email, phone,
  status [attending|not_attending],
  guest_count, note,
  calendar_added, created_at
)

-- Gifts
gifts (
  id, event_id, guest_id,
  amount, payment_method [paystack|transfer],
  paystack_reference, status [pending|confirmed],
  created_at
)
```

## Key Differentiators vs Zola/The Knot

1. Built for Nigeria — culturally resonant AI designs
2. WhatsApp-first sharing (not email)
3. Paystack gifting + bank transfer (not Stripe/Venmo)
4. Works on 3G (lightweight, mobile-first)
5. MailerLite reminders in Nigerian English tone
6. Burial brochures (completely unserved market)
7. No watercolor. Ever.
